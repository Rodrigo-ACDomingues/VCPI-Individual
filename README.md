# VCPI-Individual
Repositório criado para o projeto individual de Visao por Computador e Processamento de Imagem, ano letivo 2025/2026

Os notebooks estão separados por quantidade de imagens que usam do dataset original, o notebook é capaz de fazer o download do dataset Cifar-10 para a diretoria em que o notebook está a
correr para poder posteriormente usar as imagens. Atualmente o download está configurado para download=False mas na célula 3 de cada notebook é possível alterar este valor para True.
